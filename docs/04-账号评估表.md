# 04-账号评估表

## 1. 评估维度

| 维度 | 说明 | 评分提示 |
|---|---|---|
| 传播效率 | 标题、选题、人设、话题是否容易让人停留和转发 | 1-10 |
| 获客效率 | 是否能筛出明确需求人群，是否有清晰下一步 | 1-10 |
| 信任效率 | 是否建立专业、可信、可持续的服务认知 | 1-10 |
| 合规可持续性 | 表达和承接是否远离高风险交易模式 | 1-10 |
| 企业客户适配度 | 是否能吸引老板、采购、技术负责人、财务、安全等角色 | 1-10 |

## 2. 三类账号模型评分

| 模型 | 传播效率 | 获客效率 | 信任效率 | 合规可持续性 | 企业客户适配度 |
|---|---:|---:|---:|---:|---:|
| 行业认知型 | 8-9 | 5-6 | 8 | 7-8 | 8 |
| 普通人入局型 | 6-7.5 | 7 | 5-6 | 3-4.5 | 5 |
| 价格清单型 | 5-6 | 8-9 | 3-4 | 1-2.5 | 3 |

## 3. 内容模型识别

### 行业认知型信号

- 经常讲趋势、市场、定价、合规、出海、资质。
- 标题偏判断和拆解。
- 不一定每条都强承接。
- 更容易建立长期专业感。

### 普通人入局型信号

- 经常出现普通人、新手、入局、副业、系列课。
- 标题代入感强。
- 容易引导关注、资料、服务号。
- 合规风险取决于承接到什么服务。

### 价格清单型信号

- 标题直接出现模型、价格、倍率、低价、可用性。
- 置顶或简介里有明确联系入口。
- 交易意图强。
- 如果涉及充值、测试账号、逆向、共享 key，风险很高。

## 5. 示例诊断

```text
账号类型：普通人入局型
核心套路：用普通人、副业、系列课降低理解门槛，引导用户领取资料或咨询。
传播优势：标题代入感强，适合吸引新手和中小团队。
获客优势：承接动作较清楚，容易触发咨询。
信任短板：容易被理解为机会盘或套利项目。
合规风险：如果出现卖 token、充值、中转站、低价通道、测试账号等表达，风险较高。
值得学习：低门槛解释、系列化结构、资料承接。
必须删除：副业暴利、普通人卖 token、私域拿资源。
合规改写方向：中小团队 API 采购入门、模型调用成本治理、BYOK 管理、用量审计模板。
```
