# 抖音 AI 账号内容合规研究

这是一份面向 AI API、Token、模型调用服务类短视频账号的内容研究包。它把若干抖音账号的外显内容套路拆成可复用的增长结构，同时把高风险表达改写为更可持续的企业服务方向：API 采购、BYOK 网关、模型路由、成本治理和用量审计。

## 研究边界

本项目只基于公开可见的账号信息、标题方向、简介、置顶内容和表层传播信号进行内容策略分析。

本项目不包含：

- 后台播放量、完播率、转粉率、私信量、成交量或客单价。
- 完整视频口播原文、评论区、私域聊天记录或真实交易链路。
- 对任何账号实际经营行为、法律状态或平台违规事实的最终认定。
- 中转、代充、共享账号、共享 key、余额池、反向代理转售、规避限制等操作路径。

所有评分和判断都是内容侧推断，用于比较传播、线索、信任和合规风险，不代表真实商业结果。

## 核心结论

这些账号真正值得学习的不是“卖资源”本身，而是三种内容能力：

1. 行业框架能力：把 API、Token、模型调用讲成市场、成本、采购、合规和治理问题。
2. 低门槛解释能力：把复杂技术话题讲给老板、采购、运营、开发和中小团队听。
3. 清单化承接能力：用检查表、成本表、审计表、方案模板承接明确需求。

长期更可持续的方向不是低价、倍率、充值、测试账号或灰色通道，而是：

- 企业 API 采购评估
- BYOK 网关和客户自有 key 管理
- 多模型路由和调用策略
- 成本治理、预算控制和账单透明化
- 用量审计、权限隔离和风险留痕
- 供应商尽调、SLA、数据处理和退出机制评估

## 内容模型速览

| 内容模型 | 典型打法 | 优点 | 风险 | 合规改造方向 |
|---|---|---|---|---|
| 行业认知型 | 趋势、合规、资质、定价、出海、市场体量 | 信任强、传播面广、长期品牌好 | 承接不够直接 | API 采购、供应商尽调、成本治理 |
| 普通人入局型 | 普通人、副业、系列课、资料承接 | 痛点强、门槛低、关注和私信动机强 | 容易滑向套利和高承诺 | 中小团队 API 入门、采购风险、调用成本 |
| 价格清单型 | 模型名、价格感、表格、联系入口 | 询盘直接、需求筛选快 | 合规风险最高、企业信任弱 | 成本测算、SLA 对比、用量审计模板 |

## 推荐使用方法

如果你在做 AI API、模型调用、企业 AI 接入、模型网关、AI 成本治理相关内容，可以按下面流程使用：

1. 用 [docs/01-研究报告.md](docs/01-研究报告.md) 理解三类账号为什么有效。
2. 用 [docs/02-合规改写手册.md](docs/02-合规改写手册.md) 把高风险话术改成企业服务表达。
3. 用 [docs/03-选题脚本库.md](docs/03-选题脚本库.md) 直接规划短视频选题。
4. 用 [templates/内容评估表.md](templates/内容评估表.md) 拆解新账号或竞品内容。
5. 用 [templates/合规审查清单.md](templates/合规审查清单.md) 在发布前做自查。

## 一句话原则

可以学习“痛点标题、风险拆解、系列化表达、清单化承接”，但不要学习“低价交易、代充充值、共享 key、测试账号、规避限制、私域拿资源”。
